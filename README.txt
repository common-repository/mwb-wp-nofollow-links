== Description ==

This plugin simply adds rel=”nofollow” and alt=”post_name” attributes to the links embedded over image URLs’ of your website posts, pages, and menus.

You can also segregate the internal and external links of your website through this. It is designed to give you complete control of linking for SEO purposes.


= Main Features =

*Add rel=”nofollow” attribute to external links of your website posts/pages/menus.
*Add alt=”post_name” attribute to the img tags.
*Attributes can be added and removed with ease.
*It can be used to add attributes to both internal and external links without hassle.
*Support custom post type content.
*Improves the SEO ranking of your website.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Wordpress Nofollow Links, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Wordpress Nofollow Links" and click Search Plugins. Once you've found our Wordpress Nofollow Links plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".


= Manual installation =

The manual installation method involves downloading our Wordpress Nofollow links and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).


= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
